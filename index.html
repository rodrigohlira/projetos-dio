Tags para formatação de texto

< Voltar >

Durante os estudos para formatação de texto, a primeira que aprendi foi a tag 'P', que gera os parágrafos de forma ordenada. Automaticamente ela irá agrupar os texto e pular uma linha para o outro bloco do HTML. A próxima tag é o h1 até h6. Com ela conseguimos criar títulos e subtítulos para a nossa página WEB.

A próxima Tag é Mark. Com ela é possível marcar uma parte do texto com uma determinada cor. Muito parecido com um marcador de texto manual da cor amarela. A tag 'i' deixa o texto como italico. A tag 'u' deixa o texto com um tracejado embaixo do texto. A tag 'strong' deixa o texto em negrito.

Temos também o exemplo da tag 'sub' e 'sup', cuja função é rebaixar e subir uma parte do texto, exemplo: H2O, 23 = 2 elevado a três. A tag 'Blockquote' cria uma citação longa de um autor externo, deixando o texto bem destacado em relação ao parágrafo. A tag 'del' risca o texto, isso funciona em situações que precisamos explicar que tal afirmação já não é mais válida.
